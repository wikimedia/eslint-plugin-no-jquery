# no-grep

Disallows the [`$.grep`](https://api.jquery.com/jQuery.grep/) utility. Prefer `Array#filter`.

📋 This rule is enabled as a warning in `plugin:no-jquery/all`.

## Rule details

❌ Examples of **incorrect** code:
```js
$.grep();
```

✔️ Examples of **correct** code:
```js
grep();
'test'.grep();
'test'.grep;
```
