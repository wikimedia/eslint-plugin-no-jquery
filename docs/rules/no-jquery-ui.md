# no-jquery-ui

Disallows the use of jQuery UI [widgets, interactions, effects and utilities](https://api.jqueryui.com/).

## Rule details

❌ Examples of **incorrect** code:
```js
$div.button();
$( 'div' ).button();
$div.draggable();
$( 'div' ).draggable();
$div.slider();
$( 'div' ).slider();
```

✔️ Examples of **correct** code:
```js
button();
div.button();
$.button();
draggable();
div.draggable();
$.draggable();
slider();
div.slider();
$.slider();
$div.addClass( 'foo' );
$div.removeClass( 'foo' );
$div.toggleClass( 'foo' );
$div.show();
$div.hide();
$div.toggle();
$div.position();
$div.focus();
```
