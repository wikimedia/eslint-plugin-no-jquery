# no-browser

Disallows the [`$.browser`](https://api.jquery.com/jQuery.browser/) property.

📋 This rule is enabled as a warning in `plugin:no-jquery/deprecated-1.3`.

## Rule details

❌ Examples of **incorrect** code:
```js
$.browser;
$.browser.style;
```

✔️ Examples of **correct** code:
```js
browser;
a.browser;
browser.foo;
a.browser.foo;
$.browsers;
```
