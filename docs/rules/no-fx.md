# no-fx

Disallows `$.fx`.

📋 This rule is enabled in `plugin:no-jquery/slim`.

## Rule details

❌ Examples of **incorrect** code:
```js
$.fx;
$.fx.interval;
$.fx.off;
$.fx.speeds.slow;
$.fx.start();
```

✔️ Examples of **correct** code:
```js
fx;
fx.interval;
a.fx;
```
