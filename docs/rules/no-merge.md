# no-merge

Disallows the [`$.merge`](https://api.jquery.com/jQuery.merge/) utility. Prefer `Array#concat`.

📋 This rule is enabled as a warning in `plugin:no-jquery/all`.

## Rule details

❌ Examples of **incorrect** code:
```js
$.merge();
```

✔️ Examples of **correct** code:
```js
merge();
'test'.merge();
'test'.merge;
```
