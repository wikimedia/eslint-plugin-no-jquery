# no-internal-array-methods

Disallows the `.push`/`.sort`/`.splice` methods. Prefer `[].push.call`/`[].sort.call`/`[].splice.call`.

📋 This rule is enabled as a warning in `plugin:no-jquery/recommended`.

📋 This rule is enabled as an error in `plugin:no-jquery/deprecated-4.0`.

## Rule details

❌ Examples of **incorrect** code:
```js
$( 'div' ).push( div );
$div.push( div );
$( 'div' ).first().push( div );
$( 'div' ).append( $( 'input' ).push( div ) );
$( 'div' ).sort();
$div.sort();
$( 'div' ).first().sort();
$( 'div' ).append( $( 'input' ).sort() );
$( 'div' ).splice( 0, 1 );
$div.splice( 0, 1 );
$( 'div' ).first().splice( 0, 1 );
$( 'div' ).append( $( 'input' ).splice( 0, 1 ) );
```

✔️ Examples of **correct** code:
```js
push();
[].push();
div.push();
div.push;
sort();
[].sort();
div.sort();
div.sort;
splice();
[].splice();
div.splice();
div.splice;
```
