# no-css-props

Disallows the `$.cssProps` property.

📋 This rule is enabled as an error in `plugin:no-jquery/deprecated-4.0`.

📋 This rule is enabled as a warning in `plugin:no-jquery/deprecated-3.7`.

## Rule details

❌ Examples of **incorrect** code:
```js
$.cssProps;
$.cssProps.float;
```

✔️ Examples of **correct** code:
```js
cssProps;
a.cssProps;
cssProps.foo;
a.cssProps.foo;
```
